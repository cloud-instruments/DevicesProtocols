========================================================================
    CONSOLE APPLICATION : ciupClientTest-csc Project Overview
========================================================================

ciupClientTest-csc is a test client example application written on top of
ciupClient.dll

Implements the basic functions of a typical ciup client side application

Main ciup dll interaction is in the NativeMethods.cs (P/Invoke)

Basic use (Connect to a server, ask for server data, listen for incoming data points): 

ciupClientTest-csc.exe serverIP serverPort

to see on line help:

ciupClientTest-csc.exe -h
