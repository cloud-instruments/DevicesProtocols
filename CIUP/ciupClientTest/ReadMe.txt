========================================================================
    CONSOLE APPLICATION : ciupClientTest Project Overview
========================================================================

ciupClientTest-csc is a test client application written on top of ciupClient.dll
Implements the basic functions of a typical ciup client siade application

Basic use (Connect to a server, ask for server data, listen for incoming data points): 

ciupClientTest.exe serverIP serverPort

to see on line help:

ciupClientTest.exe -h
