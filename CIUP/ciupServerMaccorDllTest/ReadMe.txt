========================================================================
    CONSOLE APPLICATION : ciupServerMaccorDllTest Project Overview
========================================================================

A simple emulator application for ciupServerMaccorDll
Can be used to simulate the ciupServer in maccor mode

The application calls the dll feeding it with data as the Maccor sw is expected to do

complete data path:

ciupServerMaccorDllTest -> ciupServerMaccorDll -> ciupServer -> ciupClientDll -> ciupClientApp

ciupServer must run in maccor mode
